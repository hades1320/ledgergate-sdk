# Contributing

Thanks for your interest in contributing! 🎉\
This project welcomes contributions in the form of bug reports,
documentation improvements, and code changes.

------------------------------------------------------------------------

## How to Contribute

### 1. Fork & Clone

Fork the repository and clone it locally.

### 2. Install Dependencies

``` bash
npm install
```

### 3. Development Workflow

-   Follow the existing folder structure
-   Write clear, readable code
-   Keep changes focused and small
-   Add tests where applicable

### 4. Commit Guidelines

-   Use clear, descriptive commit messages
-   Prefer conventional commits if possible

### 5. Pull Requests

-   Explain **what** and **why**, not just **how**
-   Link related issues if applicable
-   Ensure CI passes before requesting review

------------------------------------------------------------------------

## Code Style

-   Formatting and linting are enforced using **BiomeJS**
-   TypeScript is the source of truth
-   Prefer explicit types for public APIs

------------------------------------------------------------------------

## Reporting Issues

Please include: - Clear reproduction steps - Expected vs actual
behavior - Environment details (Node version, OS)

------------------------------------------------------------------------

## Scope of Contributions

### Welcome

-   Bug fixes
-   Documentation improvements
-   Performance improvements
-   New adapters (SDK)

### Out of Scope

-   Breaking API changes without discussion
-   Payment-handling logic
-   Features outside the project's stated goals

------------------------------------------------------------------------

## Code of Conduct

Be respectful and constructive.\
Harassment or abusive behavior will not be tolerated.
